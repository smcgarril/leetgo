# leetgo
A learning exercise in Go programming and web development.  

### Quick Start
(Assumes an [installation of Go](https://go.dev/doc/install))

1. Clone the repo
  ```
  $ git clone https://github.com/smcgarril/leetgo.git
  $ cd leetgo
  ```

2. Run the program:
  ```
  $ go run .
  ```

3. View in browser
  http://localhost:8080
